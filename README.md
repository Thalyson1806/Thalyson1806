## 📦 Projetos em destaque

### [file-up-service](https://github.com/Thalyson1806/file-up-service)
Plataforma completa de upload e processamento assíncrono de arquivos integrada ao Google Cloud Platform. Recebe arquivos via API REST (FastAPI), armazena no Cloud Storage, enfileira tarefas no Cloud Tasks, registra metadados no PostgreSQL e grava todos os eventos de auditoria no BigQuery. Frontend em Angular com drag & drop e atualização de status em tempo real. `Python` `FastAPI` `GCP` `Cloud Tasks` `Cloud Storage` `BigQuery` `PostgreSQL` `Angular` `Docker`

### [epi-manager](https://github.com/Thalyson1806/epi-manager)
Sistema corporativo completo para gestão eletrônica de EPIs com identificação biométrica via DigitalPersona, fluxo de entrega guiado em 4 etapas, geração automática de Ficha EPI15 no padrão NR-6 com assinatura biométrica, alertas de vencimento por e-mail e controle de acesso por perfil. Bridge biométrica em Python (libfprint). `C# .NET 8` `React 18` `TypeScript` `Python` `PostgreSQL` `Docker` `Clean Architecture`

### [LogiMaster](https://github.com/Thalyson1806/LogiMaster)
Sistema logístico full stack cobrindo o fluxo de expedição do início ao fim: EDI (EDIFACT), romaneios com fluxo de status, faturamento, rastreamento de motoristas em tempo real via SignalR, mapa Leaflet, geração de PDF e app mobile com GPS. `C# .NET 8` `Next.js 16` `React Native/Expo` `TypeScript` `PostgreSQL` `SignalR` `Clean Architecture`

### [monitor-web](https://github.com/Thalyson1806/monitor-web)
Sistema de monitoramento de servidores self-hosted com agente .NET multiplataforma (Windows/Linux), dashboard em tempo real via SignalR, alertas automáticos de CPU/RAM/Disco, notificações por e-mail e modo wall para TV em sala de servidores. `C# .NET 8` `React 18` `TypeScript` `PostgreSQL` `SignalR` `Docker`

### [TaskManagerAPI](https://github.com/Thalyson1806/TaskManagerAPI)
API REST de gerenciamento de tarefas com autenticação JWT, sistema de roles (Admin/User), CRUD isolado por usuário e hash BCrypt. `C# .NET 8` `MySQL` `Entity Framework Core` `Swagger`
