## 📦 Projetos em destaque

### [epi-manager](https://github.com/Thalyson1806/epi-manager)
Sistema corporativo completo para gestão eletrônica de EPIs com identificação biométrica via DigitalPersona U.are.U (libfprint no Linux, SDK nativo no Windows), fluxo de entrega guiado em 4 etapas, geração automática de Ficha EPI15 no padrão NR-6 com assinatura biométrica impressa por linha, alertas de vencimento por e-mail (job diário via SMTP), painel de pendências e controle de acesso por perfil. `C# .NET 8` `React 18` `TypeScript` `PostgreSQL` `Docker` `Clean Architecture`

### [LogiMaster](https://github.com/Thalyson1806/LogiMaster)
Sistema logístico full-stack cobrindo o fluxo de expedição do início ao fim: entrada de pedidos via EDI (EDIFACT DELFOR/DELJIT/RND), romaneios com fluxo de status, faturamento, rastreamento de motoristas em tempo real via SignalR + mapa Leaflet, geração de etiquetas e romaneios em PDF, controle de armazém e app mobile com GPS. `C# .NET 8` `Next.js 16` `React Native / Expo` `TypeScript` `PostgreSQL` `SignalR` `Clean Architecture`

### [monitor-web](https://github.com/Thalyson1806/monitor-web)
Sistema de monitoramento de servidores self-hosted com agente .NET multiplataforma (Windows + Linux). Coleta CPU, RAM, disco e rede a cada poucos segundos, atualiza o dashboard ao vivo via SignalR, dispara alertas quando métricas ultrapassam 90% ou servidor fica offline, notificações por e-mail e modo wall para TV em sala de servidores. `C# .NET 8` `React 18` `TypeScript` `PostgreSQL` `SignalR` `Docker`

### [TaskManagerAPI](https://github.com/Thalyson1806/TaskManagerAPI)
API REST de gerenciamento de tarefas com autenticação JWT, sistema de roles (Admin/User), CRUD isolado por usuário e hash BCrypt. `C# .NET 8` `MySQL` `Entity Framework Core` `Swagger`

### [api-pedidos](https://github.com/Thalyson1806/api-pedidos)
API REST de pedidos desenvolvida como desafio técnico. CRUD completo com mapeamento de campos e integração MongoDB. `Node.js` `Express` `MongoDB`
