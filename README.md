## 🚀 Projetos em Destaque

### 🗂️ [file-up-service](https://github.com/Thalyson1806/file-up-service)
Plataforma de upload e processamento assíncrono de arquivos integrada ao GCP. Recebe arquivos via FastAPI, armazena no Cloud Storage, enfileira no Cloud Tasks com retry automático, registra metadados no PostgreSQL e audita eventos no BigQuery com histórico imutável consultável via SQL. Frontend Angular com drag & drop e status em tempo real.

`Python` `FastAPI` `GCP` `Cloud Tasks` `BigQuery` `PostgreSQL` `Angular` `Docker`

---

### 🦺 [epi-manager](https://github.com/Thalyson1806/epi-manager)
Sistema corporativo de gestão eletrônica de EPIs com identificação biométrica (DigitalPersona), fluxo guiado em 4 etapas, geração de Ficha EPI15 no padrão NR-6, alertas por e-mail e controle de acesso por perfil.

`C# .NET 8` `Python` `React 18` `TypeScript` `PostgreSQL` `Docker` `Clean Architecture`

---

### 🚛 [LogiMaster](https://github.com/Thalyson1806/LogiMaster)
Sistema logístico full stack com EDI (EDIFACT), romaneios, faturamento, rastreamento de motoristas em tempo real via SignalR, geração de PDF, controle de armazém e app mobile com GPS.

`C# .NET 8` `Next.js` `React Native` `Expo` `TypeScript` `PostgreSQL` `SignalR` `Clean Architecture`

---

### 🖥️ [monitor-web](https://github.com/Thalyson1806/monitor-web)
Monitoramento de servidores self-hosted com agente .NET multiplataforma, dashboard ao vivo via SignalR, alertas automáticos de CPU/RAM/Disco e modo wall para exibição em TV na sala de servidores.

`C# .NET 8` `React 18` `TypeScript` `PostgreSQL` `SignalR` `Docker`

### ✅ [TaskManagerAPI](https://github.com/Thalyson1806/TaskManagerAPI)
API REST de gerenciamento de tarefas com autenticação JWT, sistema de roles (Admin/User), hash de senhas com BCrypt, arquitetura em camadas e documentação Swagger interativa.
